# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 提供完整万字文档
- 参考导入调试视频、讲解视频及开发工具安装包

## 适用场景
- 运动服装销售行业
- 企业内部信息化管理

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
